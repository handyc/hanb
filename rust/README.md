# Rust Implementation of Hanb

## Run

You must have rust and cargo installed.

* Clone this repo:

```
git clone https://github.com/handyc/hanb.git
```

* cd into rust

```
cd rust
```

* Build

```
cargo build --release
```

* Run

```
./target/release/hanb xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

Enjoy hanb
